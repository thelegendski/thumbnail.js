Q&A
how do i use fonts?

due to the need to load, it is highly recommended that you place your thumbnail inside the draw function. to use a specific font(s), simply add a link tag that imports your fonts at the top of the header as you normally would.

 do i need to know
how to use PJS? no, you do not NEED to, but ski.js relies upon the beginner-friendliness of PJS in the utility an' names of functions. that said, you don't need to know how to animate or create games in PJS to use ski.js for thumbnails. how do i load
images?

use the loadImage function to load an image by inserting your image source as the first argument. proceed to use loadImage as you would getImage. you'll need to use a draw function as the image needs to load.
